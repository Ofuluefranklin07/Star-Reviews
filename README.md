# Star Reviews

A modern web application for displaying and managing star-based reviews built with React and Vite.

**Live Demo:** [https://star-reviews.vercel.app](https://star-reviews.vercel.app)

---

## Overview

Star Reviews is a fast, responsive application designed to showcase product or service reviews with star ratings. Built with modern web technologies, it provides an intuitive user experience for both viewing and submitting reviews.

---

## Features

- ⭐ Star-based rating system
- ⚡ Lightning-fast performance with Vite
- ⚛️ Built with React for dynamic UI
- 📱 Fully responsive design
- 🔧 Development-ready with ESLint configuration
- 🚀 Production-optimized build process

---

## Tech Stack

- **Frontend Framework:** React 19
- **Build Tool:** Vite 8
- **Styling:** CSS
- **Runtime:** ES Modules

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Ofuluefranklin07/Star-Reviews.git
   cd Star-Reviews
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

### Development

Start the development server with hot module replacement (HMR):

```bash
npm run dev
```

The application will be available at `http://localhost:5173`

### Build for Production

Create an optimized production build:

```bash
npm run build
```

### Preview Production Build

Preview the production build locally:

```bash
npm run preview
```

### Linting

Run ESLint to check code quality:

```bash
npm run lint
```
